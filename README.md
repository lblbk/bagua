# Bagua-Taro

## quick start

1. 两个分支需要不同安装包，所以推荐安装方案,分别放到两个文件夹内

```
git clone https://github.com/lblbk/bagua bagua
git clone https://github.com/lblbk/bagua bagua-taro
```

2. 切换 taro 进行开发

```
cd bagua-taro
git checkout taro
npm install
```

3. [注]切换到 main 也是一样操作

```
cd bagua
git checkout main   # 通常默认 main
npm install
```

4. 运行

```bash
npm run dev:h5

npm run dev:weapp
```
