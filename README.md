# bagua


## 安装

```
# 1. 创建 Vite 项目 (选择 React + JavaScript/SWC)
npm create vite@latest coin-toss -- --template react

# 2. 进入项目目录
cd bagua

# 3. 安装依赖
npm install

# 4. 运行
npm run dev
```

## 测试

```
# powershell 
$env:VITE_VOLCENGINE_API_KEY=""

# linux/mac
export VITE_VOLCENGINE_API_KEY=""
```

## AI 解读

添加环境变量

`VITE_DASHSCOPE_API_KEY` 或者 `VITE_VOLCENGINE_API_KEY`
